02_08 - Como o backend e o frontend se comunicam?

Através de API = Aplication Programing Interface
Interface = Como o software ou equipamento se comunica com o mundo
Interface do backend = Interface de programação de aplicação (API)
URL / Endpoint = Pega no frontend para acessar o backend
Para API Funcionar = Normas e protocolos padrões.
Como se fosse o garçom. 
Cliente pede comida X
Garçom leva o pedido para a cozinha (request)
Cozinha entrega o pedido ao garçom
Garçom leva o pedido ao  cliente (response)
Faz comunicação entre backend e backend também
API = Conjunto de normas que possibilita a comunicação entre plataformas utilizando padrões e protocolos, utilizando a mesma regra

02_11 - O começo de tudo

Input = Problema a ser resolvido
Output = Problema resolvido
Código = Onde é feito o programa (ações para resolver o problema)
Padrão ASCII (256) = American Standart Code for Information Interchange
Cada letra tem 8 bits (Binary Digits) e por isso 8 bits = 1 byte
Padrão Unicode (138000) = Padrão para outros idiomas para comunicação efetiva
